## 📞 Support & Troubleshooting

### Common Questions

**Q: How long does deployment take?**  
A: ~10-15 minutes total (most time spent training ML models)

**Q: Can I skip model training?**  
A: Yes, use: `sudo bash setup_rocky_linux.sh --skip-models`

**Q: Where are the generated artifacts?**  
A: See [PROJECT_REPORT.md](PROJECT_REPORT.md) → "Key Artifacts & Deliverables"

**Q: How do I check if services are running?**  
A: `sudo systemctl status prefect-server.service data-pipeline.service mlflow-ui.service`

### Logs & Debugging

```bash
# View real-time logs
sudo journalctl -u data-pipeline.service -f    # Pipeline logs
sudo journalctl -u prefect-server.service -f   # Prefect logs
sudo journalctl -u mlflow-ui.service -f        # MLflow logs

# Check service status
sudo systemctl status prefect-server.service
sudo systemctl status data-pipeline.service
sudo systemctl status mlflow-ui.service
```
